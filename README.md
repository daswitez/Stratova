# Stratova - Plataforma de Simulación Empresarial

## Vista del Estudiante

Plataforma de simulación empresarial por cohortes donde equipos compiten durante varios ciclos tomando decisiones estratégicas en diferentes módulos de gestión.

## Características Principales

### Sistema de Simulación
- **Ciclos de decisión**: Ventanas temporales abiertas/cerradas para toma de decisiones
- **Trabajo en equipo**: Cada miembro tiene un rol específico (Finanzas, Marketing, RRHH, Operaciones)
- **Bloqueo de decisiones**: Las decisiones se bloquean al enviarlas, asegurando integridad
- **IA integrada**: Predicción de demanda, detección de anomalías, recomendaciones estratégicas

### Módulos Implementados

1. **Workspace Ejecutivo** - Panel principal con KPIs y alertas críticas
2. **Centro de Decisiones** - Interfaz para tomar decisiones por módulo con impacto estimado
3. **Módulo Mercado** - Análisis de segmentación, competencia y predicción de demanda
4. **Módulo Finanzas** - Flujos de caja, VAN, TIR, análisis de escenarios
5. **Módulo RRHH** - Estructura organizacional, productividad, detección de ineficiencias
6. **Ranking** - Posición competitiva y comparación con otros equipos
7. **Análisis General** - Consolidación de KPIs y benchmarking inteligente
8. **Perfil** - Información del usuario y estadísticas de desempeño

## Diseño: Liquid Glass

Sistema de diseño corporativo minimal con efecto "vidrio líquido" inspirado en Apple:

- **Colores**: Paleta sobria con azul profundo como accent (#1D4ED8 light / #3B82F6 dark)
- **Tipografía**: Inter para UI, JetBrains Mono para números y métricas
- **Componentes Glass**: Cards con backdrop blur, bordes sutiles y sombras elegantes
- **Animaciones**: Motion suaves y con propósito (stagger, hover micro-lift, count-up)
- **Light/Dark Mode**: Toggle nativo con transiciones suaves

## Datos Simulados

La aplicación incluye datos mock realistas:
- Equipo Alpha en posición #2 de 5 equipos
- Ciclo 3 de 8 en progreso (ventana abierta)
- KPIs financieros, de mercado y operacionales
- Alertas y eventos del ciclo
- Historial de decisiones con auditoría
